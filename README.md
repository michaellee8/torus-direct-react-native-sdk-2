# torus-direct-react-native-sdk-2

React Native SDK for Torus CustomAuth

## Installation

```sh
npm install torus-direct-react-native-sdk-2
```

## Usage

```js
import { multiply } from "torus-direct-react-native-sdk-2";

// ...

const result = await multiply(3, 7);
```
